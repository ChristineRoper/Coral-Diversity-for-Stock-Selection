# Selecting Acropora millepora with predicted thermotolerance for restoration practices poses minimal risk to local genetic diversity on the Great Barrier Reef

Assessment of coral photophysiological phenotypic performance  via a rapid, high-thoroughput phenotyping system, alongside analysis of genetic diversity measures to determine if phenotypic and genetic diversity align.

Version 1

# Files & Folders
## `Phenotype Analysis`
Data and code for quantifying Fq´/Fm´ effective dose 50 (ED50) following heat stress.  
See `Phenotype Analysis/README` for overview of files.

## `Genetic Analysis`
Data and code for SNP quality control, filtering, visualisation and analysis to infer population structure, genetic distance, expected heterozygosity and allelic richness.  
See `Genetic Analysis/README` for overview of files.

## `Temperature`
Visualise and calculate temperature profiles from AQUA MODIS satellite data for each reef.
* Download AQUA MODIS Sea Surface Temperatur,e 2021-01-01 to 2021-01-08, Level 3, 8 Day into `AQUA_MODIS.L3m.8D.SST.x_sst`   
https://podaac.jpl.nasa.gov/dataset/MODIS_AQUA_L3_SST_MID-IR_DAILY_4KM_NIGHTTIME_V2019.0
* Run `Temperature.r` to characterise temperature profiles and generate plots.
